# dotfiles

My personal configuration for Linux applications.
Managed with [chezmoi](https://www.chezmoi.io).
Uses Bitwarden for secrets management.

## Installation

Run this command in your terminal:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Alejojooo/dotfiles/main/install.sh)"
```

This will install and configure everything.
